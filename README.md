Proyecto de la segunda unidad para el curso de Bases de Datos Avanzadas. 
En este proyecto se utilizará JPA con Hibernate para programar y mapear la base de datos con la que trabajará el sistema de CampusConnect, que busca unir a los estudiantes de una universidad mediante una red social.
